Engineering Skills

TypeScript

My favorite language, combining static typing with flexibility to build clean, scalable, and maintainable applications. Proficient in leveraging advanced features like generics and type guards for robust development.

React

Extensive experience with React, my go-to library for building dynamic, performant, and user-friendly interfaces. Skilled in designing reusable components, managing state effectively with hooks or context, and optimizing performance for seamless user experiences.

NodeJS

Node.js offers a vast ecosystem and solid performance, but its biggest advantage is enabling the use of the same language, TypeScript, on both frontend and backend. This consistency simplifies development, improves collaboration, and speeds up the delivery of scalable applications.

Functional Programming

I’ve learned both OOP and FP, but lately, I’ve come to appreciate FP for its simplicity and effectiveness. Immutability, higher-order functions, and declarative patterns, when applied correctly, make it easier to build clean, scalable, and maintainable code in TypeScript.

CI/CD

Implemented and optimized continuous integration and deployment pipelines to ensure fast, reliable, and seamless deliveries. Automated testing, builds, and deployments on cloud environments (GCP, Kubernetes) while embracing a DevOps approach.

Infra As Code
